# MindMate AI 🤖

##  Problem
Students struggle with consistency, stress, and planning.

##  Solution
MindMate AI is a smart study assistant that helps students:
- Chat support
- Study planner
- Progress tracking
- Focus timer
- Login system

##  Features
- Personalized study plans
- Daily streak tracking
- Pomodoro timer
- User authentication

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
